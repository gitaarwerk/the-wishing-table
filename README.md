# The Wishing Table
TheWishingTable is a World of Warcraft addon that provides an notification when you put down consumables for your fellow players

## Installation
- Download the latest version of the Wishing Table addon from (https://legacy.curseforge.com/wow/addons/the-wishing-table)[the CurseForge page].
- Extract the downloaded archive and copy the TheWishingTable folder to your WoW Interface/AddOns directory.
- Launch WoW and log in to your character.
- Open the AddOns tab in the character selection screen and make sure TheWishingTable is enabled.

## Usage
TheWishingTable automatically monitors any resurrection spells cast by you in your group or raid. 

To access the settings, type `/twt` or `/wishingtable` in the chat window or use the Blizzard addon settings interface (Esc -> Interface -> AddOns -> TheWishingTable).

In the settings, you can also configure which spells to monitor and enable/disable chat notifications of resurrection spells.
